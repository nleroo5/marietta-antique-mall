- Less reliable, depends on user's email client
- Not recommended for file uploads

**Option 3: Third-Party Form Service**
- Formspree, Basin, or similar
- Handles file uploads automatically
- Sends to specified email
- May have monthly limits on free tier

### Form Data Storage (Optional)

**Consideration:** Store applications in database
- Benefits: Track applications, search history, prevent duplicates
- Implementation: Add to API route, save to database before emailing
- Database: Vercel Postgres, Supabase, or similar
- GDPR Compliance: Include privacy policy, data retention policy

**Not Required for MVP** - Email-only is acceptable for initial launch

---

### File Upload Handling

**Requirements:**
- Max file size: 10MB per file
- Accept: JPG, PNG, PDF
- Multiple files (3-5 recommended)
- Virus scanning (if using cloud storage)
- Temporary storage until email sent
- Auto-delete after successful send

**Implementation Options:**
1. **Direct Base64 encoding** (for small files)
   - Encode on client, attach to email
   - Simple but limited to smaller files

2. **Cloud storage** (for larger files)
   - Upload to Vercel Blob, AWS S3, Cloudinary
   - Include links in email
   - More scalable

3. **Temporary server storage**
   - Upload to /tmp directory
   - Attach to email
   - Clean up after send
